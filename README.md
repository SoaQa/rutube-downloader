# Rutube Downloader

Локальное GUI-приложение на Python для скачивания видео через `yt-dlp`.

Приложение изначально делалось под ссылки Rutube, но `yt-dlp` также поддерживает YouTube и многие другие видеосервисы.

## Возможности

- Несколько ссылок на видео в одном запуске.
- Добавление URL через кнопку `+`.
- Параллельное скачивание в несколько потоков.
- По умолчанию используется `4` потока, но приложение не запускает больше потоков, чем указано ссылок.
- Выбор качества видео.
- Проверка доступных форматов по первой ссылке.
- Выбор базовой директории скачивания.
- Дополнительное поле `Folder name` для подпапки, например `series`.
- Скачивание плейлистов опционально.
- Опциональный режим `Ignore SSL certificate errors` для проблемных сертификатов.
- Горячие клавиши `Cmd/Ctrl + A/C/V/X`, не зависящие от русской или английской раскладки.

## Требования

- Python 3.12 или новее.
- `tkinter`, обычно идет вместе с Python.
- `ffmpeg` может понадобиться `yt-dlp` для некоторых YouTube-форматов, где видео и аудио скачиваются отдельно.

## Установка

```bash
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
```

## Запуск

```bash
.venv/bin/python app.py
```

## Как пользоваться

1. Вставьте одну или несколько ссылок в блок `Video URLs`.
2. Нажмите `+`, чтобы добавить еще одно поле URL.
3. В `Save to` выберите базовую папку скачивания. По умолчанию это `downloads` внутри проекта.
4. В `Folder name` можно указать имя подпапки. Например, `series` даст путь `downloads/series`.
5. Выберите качество.
6. При необходимости нажмите `Check`, чтобы загрузить список форматов для первой ссылки.
7. Укажите количество потоков в `Threads`.
8. Нажмите `Download`.

Пустые строки URL игнорируются.

## Директории скачивания

По умолчанию файлы сохраняются в:

```text
downloads/
```

Если указать:

```text
Folder name: series
```

файлы сохранятся в:

```text
downloads/series/
```

Можно указывать вложенные папки, например:

```text
series/season 1
```

## Зависимости

- `yt-dlp` — скачивание и получение форматов видео.
- `certifi` — актуальный bundle SSL-сертификатов.

Зависимости зафиксированы в `requirements.txt`.

## Примечания

Папка `downloads/` не добавляется в Git, чтобы случайно не загрузить скачанные видео в репозиторий.
